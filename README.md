Analog Clock using HTML, CSS, and JavaScript

A simple and clean analog clock created using HTML, CSS, and JavaScript.  
It displays the current time using rotating hour, minute, and second hands.

Features
-> Real-time rotation of clock hands
-> Smooth and accurate movement
-> Minimal and clean UI
-> Beginner-friendly project

 What I Learned
-> Using `Date()` object in JavaScript
-> Calculating rotation angles
-> CSS positioning and transformations
-> DOM manipulation
